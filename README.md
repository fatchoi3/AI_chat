# AI 챗봇 플랫폼 프로젝트

## 프로젝트 소개  
이 프로젝트는 React Native를 기반으로 한 모바일 AI 챗봇 앱과 Node.js 백엔드를 연동하여, 교회 내부에서 사용할 수 있는 맞춤형 AI 컴패니언을 개발하는 것을 목표로 해.

## 주요 기능  
- OpenAI GPT API를 활용한 AI 대화 기능  
- 실시간 메시지 송수신 및 대화 이력 관리  
- React Native 모바일 UI/UX 구현  
- Node.js와 PostgreSQL을 이용한 백엔드 API 개발  

## 기술 스택  
- React Native  
- Node.js  
- PostgreSQL  
- OpenAI API  
- WS (실시간 통신, web socket)
